appeler

The project is devoted to making it easier to learn the gender, race and ethnicity from human names, and to making easier to search for human names.

Imputing Sex

  • R Data Package
  • Infer the gender of person with a particular first name using Google image search and Clarifai

Imputing Race & Ethnicity

  • Python Package
  • R Package

Tools

  • Search a long list of names (patterns) in a large text corpus systematically and quickly
  • Deduplicate and parse list of `dirty names'
